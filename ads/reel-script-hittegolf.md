# REEL-SCRIPT: "De Nederlandse Hittegolf" — Sonty

**Doel**: 500k+ views. **Duur**: ±26 sec. **Formaat**: 9:16. **Stijl**: droge NL comedy + satisfying payoff.

## Waarom dit concept viraal kán (onderbouwd uit playbook + data)

- **Herkenbaarheid = shares** (shares/sends = hét viraal-signaal): élke Nederlander kent de hittegolf-paniek, de lege ventilator-schappen en vooral **de lakens-voor-de-ramen-fase** — dat beeld alleen al is een share-trigger ("dit zijn wij 😂")
- **Breed publiek**, niet alleen kopers: comedy reikt voorbij de doelgroep; de doelgroep (huiseigenaar 35-65) voelt de payoff
- **Buurman-element** sluit exact aan op onze sterkste conversiebron (buren 9,7%)
- **Satisfying screen-drop** als retentie-anker; **loop-einde** voor rewatches
- **Serie-potentieel**: "De Nederlandse Zomer deel 2/3..." (algoritme beloont consistentie)

**Twee audioversies**: (a) met VO zoals hierboven, (b) zonder VO — alleen sound design, zodat er in IG trending audio onder kan.

## Productieplan (pas ná akkoord op dit script)

1. **Frames**: per shot een start-frame met nano banana — Peter via character sheet (identiteit vast), buurman = nieuw mini-character-sheet, gevel/screen op basis van échte Sonty-projectfoto als referentie
2. **Clips**: 7× Seedance zónder spraak (alleen beweging + ambient), met end-frame → start-frame continuïteit (Arcads-methode)
3. **Edit**: ffmpeg — stitchen, tempo strak, harde cuts op de beats
4. **Voice-over**: ElevenLabs, Nederlandse stem (optie: de Sonty voice clone — klinkt als het merk zelf), droge comedy-timing
5. **Mix**: ambient + VO + motor-hum layeren
6. **Tekst-overlays**: advies = native in Instagram zetten (beter voor het algoritme); ik lever daarnaast een versie met ingebakken overlays als preview
7. **QA**: virality predictor + eigen frame-check, dán pas naar jou

**Kosten-indicatie**: ~250-350 Higgsfield-credits (saldo: ~600). **Tijd**: ±1-2 uur productie na akkoord.
